# 🔐 API Security Risk Analysis

## 📌 Overview

This project contains an API Security Risk Analysis performed on a public API using Postman and Browser Developer Tools.

---

## 🧰 Tools Used

* Postman
* Browser Developer Tools

---

## 🌐 API Tested

https://jsonplaceholder.typicode.com

---

## 🔍 Methodology

* Sent API requests using Postman
* Analyzed responses, headers, and status codes
* Tested authentication and authorization
* Inspected network traffic using browser DevTools

---

## ⚠️ Key Findings

* No Authentication
* No Authorization
* Excessive Data Exposure
* Weak Rate Limiting
* Input Validation Issues

---

## 🛡️ Conclusion

The API uses HTTPS for secure communication but lacks proper security controls at the application level.
